
# apt

**APT**（Advanced Package Tool）是 Linux 系统中功能强大的高级包管理器，它在 `dpkg` 的基础上引入了**网络源**和**依赖关系自动处理**机制，能根据指令自动从远程仓库下载并安装软件及其所需的所有配套组件，是 Ubuntu 用户管理系统软件更新与维护最常用的核心工具。

# dpkg

**dpkg**（Debian Package）是 Debian 及其衍生系统（如 Ubuntu）中最底层的软件包管理工具，它直接负责对 **.deb** 格式的安装包进行安装、卸载、解压及状态查询，是 APT 等高级包管理器的基石，但不具备自动从网络下载包或自动处理复杂依赖关系的能力。

```bash

dpkg -l
# 列出所有安装的

```
