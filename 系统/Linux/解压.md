# unzip

这里是 `unzip` 命令最常用的几种场景，按照使用频率排序。

首先，确保你已经安装了它：

Bash

```
sudo apt install unzip
```

## 1. 最简单的用法 (解压到当前目录)

直接把文件解压在当前所在的文件夹里。

Bash

```
unzip file.zip
```

## 2. 解压到指定文件夹 (推荐 ✨)

使用 `-d` 参数指定目录。如果目录不存在，它会自动创建。

**场景**：保持目录整洁，避免解压出一堆文件把当前目录弄乱。

Bash

```
# 将 file.zip 解压到 /home/user/data 目录中
unzip file.zip -d /home/user/data
```

## 3. 只查看，不解压

使用 `-l` (list) 参数。

**场景**：不知道压缩包里有什么，想先看一眼文件名。

Bash

```
unzip -l file.zip
```

## 4. 解决中文乱码问题 (重要 🛠️)

如果你解压的是**Windows**系统打包的中文文件名压缩包，在 Linux 下直接解压通常会显示乱码。

使用 `-O` 指定编码（通常是 `CP936` 或 `GBK`）。

Bash

```
unzip -O CP936 file.zip
```

---

### 极简速查表

|**目的**|**参数**|**示例**|
|---|---|---|
|**指定目录**|`-d`|`unzip x.zip -d ./target_dir`|
|**查看内容**|`-l`|`unzip -l x.zip`|
|**修复乱码**|`-O`|`unzip -O CP936 x.zip`|
|**覆盖不提示**|`-o`|`unzip -o x.zip` (小心使用)|

**小提示**：如果你想解压 `.tar.gz` 文件，不能用 `unzip`，而应该用 `tar -zxvf filename.tar.gz`。
