**nvm**（Node Version Manager）是一个用于管理 **Node.js** 版本的命令行工具，它允许你在同一台机器上同时安装多个不同版本的 Node.js，并能根据项目需求在这些版本之间一键切换，从而完美解决不同项目对环境版本要求不一致所导致的兼容性冲突问题。
## 安装

### Linux

```bash

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
# Linux安装

```

```bash

nvm install 20

nvm use 20

```
