# 管理端测试版部署指南 (Port 8081 外网)

本文档说明如何将 `user_admin_web` 部署到 Nginx，并通过端口转发实现外网访问。

## 1. 端口映射说明

- **外网访问端口**: 8081
- **内网 Nginx 端口**: 8080
- **转发规则**: 外网 8081 -> 内网 8080

## 2. 项目构建

在 `user_admin_web` 目录下执行构建命令：

```bash

cd user_admin_web

npm install

npm run build:test

```

构建完成后，生成的静态文件位于 `user_admin_web/dist` 目录。

## 3. Nginx 配置

测试环境使用 `user_admin_web/nginx_test.conf`。主要配置如下：

- **监听端口**: 8080
- **根目录**: `/var/www/html/admin_test`
- **后端代理**: `http://127.0.0.1:8000`

### 配置文件位置

建议将 `nginx_test.conf` 复制到 Nginx 的配置目录（如 `/etc/nginx/sites-enabled/admin_test`）。

## 4. 部署步骤

1. **创建部署目录**:

   ```bash

   sudo mkdir -p /var/www/html/admin_test

   ```

2. **同步文件**:

   将构建好的 `dist` 内容复制到部署目录：

   ```bash

   sudo cp -r dist/* /var/www/html/admin_test/

   ```

3. **应用 Nginx 配置**:

   ```bash

   sudo cp nginx_test.conf /etc/nginx/sites-enabled/admin_test

   sudo nginx -t

   sudo systemctl reload nginx

   ```

## 5. 访问地址

部署完成后，可以通过以下地址访问：

`http://服务器外网IP:8081`

## 6. 注意事项

  
- 确保服务器防火墙已放行 8080 端口。
- 测试环境的后端 API 地址默认为 `http://127.0.0.1:8000`，请确保后端服务已启动。
