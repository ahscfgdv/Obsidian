# QASystem 项目使用手册

欢迎使用 QASystem！本系统是一套集成了 AI 智能问答、知识库管理、工单处理及 OCR 识别的综合技术支持平台。

---

## 1. 角色与入口说明

| 角色       | 主要职责                 | 访问入口                  |
| :------- | :------------------- | :-------------------- |
| **普通用户** | 咨询技术问题、提交报修工单、查阅历史记录 | `user_client` 网页端     |
| **管理员**  | 维护知识库、处理工单           | `user_admin_web` 管理后台 |

---

## 2. 普通用户端操作指南 (User Client)

### 2.1 智能问答 (Chat)

![image.png](https://raw.githubusercontent.com/ahscfgdv/obsidian-images/main/test/2026/20260331164218183_2026-03-31_164218.png)

- **多轮对话**：在“智能问答”界面直接输入问题。系统支持结合上下文的连续对话。
- **参考资料预览**：
  - AI 回答后，下方会展示“参考文档”卡片。
  - **点击预览**：直接在右侧面板查看 PDF 原文，支持缩放和翻页。
- **清空会话**：点击顶部的“清空对话”按钮可开启新的讨论主题。

### 2.2 提交工单 (Ticket)

![image.png](https://raw.githubusercontent.com/ahscfgdv/obsidian-images/main/test/2026/20260331164329529_2026-03-31_164329.png)


- **快速提交**：进入“提交工单”页面。
- **要素填写**：
  - **问题简述**：简洁明了的标题。
  - **详细描述**：尽可能描述故障现象。
  - **机器号**：输入故障机器的编号（选填）。
  - **图片上传**：点击上传区域或直接粘贴截图。
- **进度追踪**：在“工单记录”中查看实时状态（待处理、处理中、已解决、已关闭）。

### 2.3 历史会话 (History)

![image.png](https://raw.githubusercontent.com/ahscfgdv/obsidian-images/main/test/2026/20260331164413933_2026-03-31_164413.png)


- 系统会自动按天对聊天会话进行归档，方便随时回顾之前的解决方案。

## 3. 管理后台操作指南 (User Admin Web)

### 3.1 知识库管理 (Knowledge)



- **文档上传与入库**：
  - 支持 `.pdf`, `.doc`, `.docx` 批量上传。
  - **状态说明**：
    - `待审核`：新上传文档，AI 暂无法检索。
    - `已发布`：审核通过，系统已完成向量化，正式进入 AI 检索池。
    - `已驳回`：文档不符合规范或内容有误。
- **分类维护**：在“分类管理”中创建树状层级，确保护理、硬件、软件等文档条理清晰。

### 3.2 知识模板生成 (Template)



- **规范化文档**：提供标准的 Word 模板填充功能。
- 管理员可输入问题背景、操作步骤及关键要点，系统自动生成符合规范的解决方案文档并上传到知识库。

### 3.3 工单闭环处理 (Ticket Management)

![image.png](https://raw.githubusercontent.com/ahscfgdv/obsidian-images/main/test/2026/20260331164921408_2026-03-31_164921.png)


- **领取与响应**：在列表中点击“去处理”按钮。
- **解决方案填写**：
  - 填写处理结果描述。
  - **上传处理文档**：若无成熟解决方案可以上传解决文档。

### 3.4 用户与安全 (User)

![image.png](https://raw.githubusercontent.com/ahscfgdv/obsidian-images/main/test/2026/20260331165010624_2026-03-31_165010.png)


- **用户维护**：管理员可以手动创建账号、启用/禁用用户、修改客户名称。
- **登录鉴权**：系统采用 JWT 校验，管理员长时间未操作会自动登出。

## 4. 常见问题 (FAQ)

**Q: 为什么上传的 PDF 在预览时显示空白？**  
A: 请检查文件是否损坏，或尝试在后台重新上传。若 PDF 只有图片而无文字，AI 检索效果会受限。

**Q: 工单提交成功后，我（用户）能修改吗？**  
A: 提交后不支持修改描述，建议在“详细描述”中一次性说清楚。若有补充，可在“对话”中告知管理员。

**Q: 为什么 AI 的回答有时不准确？**  
A: AI 的回答基于你提供的知识库文档。建议优化原始文档的描述，或在提问时提供更具体的关键词。
