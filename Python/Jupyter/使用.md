## 安装使用

```bash

pip install ipykernel 
#ipykernel是jupyter的内核，如果要使用web端，直接安装jupyter
pip install jupyter

# 运行
jupyter notebook
```

## 在vscode中使用uv虚拟环境做jupyter内核

```bash

uv venv --python 3.10
.venv\Scripts\Activate
uv add ipykernel
# 注册内核（给内核起个名字，如 "uv_llm"）
python -m ipykernel install --user --name uv_llm --display-name "UV LLM Environment"

```
